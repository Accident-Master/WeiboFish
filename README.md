# WeiboFish
模仿Mirofish制作的政务新媒体多智能体仿真平台<br>
本项目在开发过程中受到了MiroFish项目关于多智能体仿真逻辑的启发。WeiboFish在此基础上，针对政务新媒体场景进行了实证数据校准与功能迭代，特此向原作者/研究团队表示感谢。<br>
<br>
